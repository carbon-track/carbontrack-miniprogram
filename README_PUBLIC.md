# CarbonTrack Mini Program

**CarbonTrace** is a WeChat Mini Program designed specifically for the WeChat ecosystem, aiming to make carbon tracking simple, engaging, and accessible for students.

> 🚧 *This is a preliminary version. The name "CarbonTrace" is temporary and may be updated in future releases.*

---

## 🌱 About the Project

CarbonTrace helps users record their daily eco-friendly actions and convert them into measurable carbon reduction.

The Mini Program is built to:
- Encourage sustainable habits  
- Visualize carbon impact  
- Gamify environmental action  

---

## ⚙️ Features (In Progress)

- 📝 Log daily low-carbon activities  
- 📊 Calculate CO₂ reduction  
- 🏆 Carbon credits & leaderboard  
- 🎁 Reward system *(under development)*
