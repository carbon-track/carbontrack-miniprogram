# CarbonTrack 小程序开发注意事项

## 开发环境配置

为了在开发环境中正常运行小程序，请在微信开发者工具中进行以下设置：

1. 点击顶部菜单栏的「详情」
2. 选择「本地设置」
3. 勾选「不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书」选项

## 服务器地址配置

小程序目前配置使用本地开发服务器：
- 本地开发环境地址：`http://localhost:8000`

在正式发布前，请确保：
1. 修改 `app.js` 中的 `baseUrl` 为正式服务器地址
2. 在微信公众平台配置合法的服务器域名

## 其他注意事项

- 确保本地开发服务器正在运行（端口 8000）
- 对于 API 调用错误，请检查网络连接和服务器状态
- 开发完成后，记得关闭「不校验合法域名」选项进行最终测试
