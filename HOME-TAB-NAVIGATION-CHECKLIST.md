## 🐛 常见问题排查

### 问题1: Tab切换失败
**症状**: 点击功能卡片后没有切换到Tab页面
**可能原因**:
1. `type`字段设置错误
2. URL路径不正确
3. Tab页面不在tabBar配置中

**解决方案**:
1. 检查`features`数组中的`type`字段
2. 检查URL是否与`app.json`中的`pagePath`一致
3. 确认页面在`tabBar.list`中

### 问题2: 页面堆叠
**症状**: Tab页面以堆叠方式打开，而不是切换
**可能原因**: 使用了`wx.navigateTo`而不是`wx.switchTab`

**解决方案**:
1. 检查`onFeatureTap`函数中的导航逻辑
2. 确认`type === 'tab'`时使用`wx.switchTab`

### 问题3: 数据传递失败
**症状**: 点击事件没有传递`type`字段
**可能原因**: WXML中`data-type`绑定错误

**解决方案**:
1. 检查WXML中的`data-type="{{item.type || 'page'}}"`
2. 确保`item.type`在数据中定义

   ```javascript
   // 修改后:

   // 2. 恢复onFeatureTap函数

     const url = e.currentTarget.dataset.url;
     if (url) {
       wx.navigateTo({ url });
     }
   }
   
   // 3. 恢复其他按钮的导航方式
   wx.navigateTo({ url: '/pages/calculate/calculate' });
   ```
