# CarbonTrack 微信小程序

CarbonTrack是一个环保碳足迹记录与计算的微信小程序，复刻自CarbonTrack网站。

## 功能特点

- **碳足迹计算**：支持多种日常活动的碳足迹计算
- **排行榜系统**：全球、学校、好友排行榜展示
- **用户中心**：个人信息管理、积分统计、活动记录
- **主题切换**：支持浅色/深色主题自动适配系统设置
- **响应式设计**：适配各种移动设备屏幕
- **数据可视化**：碳减排成果直观展示

## 开发说明

### 环境要求

- 微信开发者工具
- Node.js
- 微信小程序账号

### 开发步骤

1. 克隆项目到本地
2. 在微信开发者工具中导入项目
3. 配置app.json中的页面路径和tabBar
4. 实现各个页面的功能

### 主题切换

小程序支持浅色和深色主题，会自动适配系统主题设置。主题相关变量在app.js中定义：

```javascript
// 浅色主题变量
{
  '--primary-color': '#34C759',
  '--background-color': '#F2F2F7',
  '--text-color': '#000000',
  // ...其他变量
}

// 深色主题变量
{
  '--primary-color': '#4ECDC4',
  '--background-color': '#1C1C1E',
  '--text-color': '#FFFFFF',
  // ...其他变量
}
```

## API 接口

项目使用了以下主要API接口：

- 用户认证：登录、注册、找回密码
- 碳足迹计算：活动类型、排放因子、计算结果
- 排行榜：全球排名、学校排名、好友排名
- 用户中心：个人信息、积分记录、活动历史

目前使用的是模拟数据，实际部署时需要连接真实后端API。
